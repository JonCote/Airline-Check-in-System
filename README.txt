ACS system execution instructions:

    1) extract the p2 tar file using the terminal command (tar -xf p2.tar.gz)
    2) navigate into the p2 folder using the terminal command (cd p2)
    3) compile the ACS system using the terminal command (make)
    4) run the ACS system using the terminal command (./ACS 'customer input file name') where the 'customer input file name' is the desired customer input file to be used.
        For inital test run a customer input file named test-input.txt was provided. Use command ./ACS test-input.txt to run the ACS system with the provided customer input file.
